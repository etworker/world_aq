# 城市坐标

`data/info/worldcities.csv`文件来自 [World Cities Database](https://simplemaps.com/data/world-cities)，是一个非常流行且易用的**全球城市基础数据库**，便于获取城市坐标、国家等信息。它的核心价值在于提供了一个标准化的、带有地理坐标（经纬度）和人口信息的城市列表，非常适合用来做**地理查询**、**数据可视化**或作为**查找最近气象站的跳板**。

其前3行内容为：

```
"city","city_ascii","lat","lng","country","iso2","iso3","admin_name","capital","population","id"
"Tokyo","Tokyo","35.6870","139.7495","Japan","JP","JPN","Tōkyō","primary","37785000","1392685764"
"Jakarta","Jakarta","-6.1750","106.8275","Indonesia","ID","IDN","Jakarta","primary","33756000","1360771077"
```

各字段含义如下：

| 字段名 (Header) | 含义          | 详细说明                                                     | 示例 (Row 1)                                                 |
| --------------- | ------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **city**        | 城市名称      | 城市的标准名称，通常包含当地语言的特殊字符（如重音符号）。用于显示。 | Tokyo                                                        |
| **city_ascii**  | ASCII 城市名  | 去除了所有特殊字符（如 é, ü, ñ）的纯英文字符串。**这是用于搜索的最佳字段**，因为它避免了编码问题。 | Tokyo (如果是慕尼黑，这里是 Munich，而 city 列可能是 München) |
| **lat**         | 纬度          | 城市的中心纬度（十进制）。+北纬，-南纬。                     | 35.6870                                                      |
| **lng**         | 经度          | 城市的中心经度（十进制）。+东经，-西经。                     | 139.7495                                                     |
| **country**     | 国家名称      | 国家的英文全称。                                             | Japan                                                        |
| **iso2**        | 2位国家代码   | **ISO 3166-1 alpha-2** 标准。常见且通用（如 CN, US, JP）。<br>⚠️ **注意**：这与 NOAA 数据中的 FIPS 代码（CH, US, JA）可能不一致，联表时需转换。 | JP                                                           |
| **iso3**        | 3位国家代码   | **ISO 3166-1 alpha-3** 标准。通常比 2 位代码更不易混淆（如 CHN, USA, JPN）。 | JPN                                                          |
| **admin_name**  | 行政区划      | 城市所属的一级行政区（省、州、都、道、府、县）。             | Tōkyō (东京都)                                               |
| **capital**     | 首都/首府状态 | 标识该城市的行政地位：<br>• primary: 国家首都<br>• admin: 一级行政区首府（省会/州府）<br>• minor: 下级行政区首府<br>• "" (空): 普通城市 | primary                                                      |
| **population**  | 人口          | 城市人口估算值。**注意**：SimpleMaps 通常使用的是**都市圈（Metropolitan）人口**，而非狭义的行政边界人口。这就是为什么东京显示 3700万，雅加达显示 3300万。 | 37785000                                                     |
| **id**          | 唯一 ID       | SimpleMaps 分配的 10 位数字唯一标识符。用于数据库关系锚定。  | 1392685764                                                   |
